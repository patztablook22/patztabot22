# patztabot22
me, but bot \
a discord human-controlled bot

# Manual
the project uses maven \
some properties are hardcoded cuz laziness (e.g. Pic path)

to compile \
`make build`

to run (add the token to `Makefile`) \
`make run`

**Commands**
  - `cc index` change channel
  - `pc` print channel
  - `lc` list channels
  - `lg` list guilds
  - `leave guild name` leave a guild
  - `msg/m message text` send a message in current channel
  - `pics` list files in Pic path
  - `pic basename` send the first match of basename.X in the Pic path
  - `quit/exit` exit the program
